# 📘 Assignment: REST APIs with FastAPI

## 🎯 Objective

Build REST APIs using the FastAPI framework to create web services that handle HTTP requests and responses. Learn to create endpoints, handle data, and serve JSON responses.

## 📝 Tasks

### 🛠️ Implement REST API with FastAPI

#### Description
Create a FastAPI application that provides RESTful endpoints for managing a simple resource, such as a list of items or user data.

#### Requirements
Completed program should:

- Install and import FastAPI and Uvicorn
- Create a FastAPI app instance
- Implement at least three endpoints: GET (list all items), POST (add new item), GET (get specific item by ID)
- Use Pydantic models for request/response data validation
- Handle basic error cases (e.g., item not found)
- Include proper status codes and JSON responses
- Run the server and test endpoints with a tool like curl or browser
